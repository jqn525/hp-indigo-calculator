Design Guidelines

File Preparation
  • Resolution: 300 DPI minimum for all images
  • Color Mode: CMYK color mode for accurate printing
  • File Format: PDF preferred, AI or EPS accepted
  • Fonts: Embed or outline all fonts
Layout Guidelines
  • Bleed: Include 0.125" bleed on all sides
  • Safe Zone: Keep important elements 0.25" from edges
  • Fold Lines: Keep text 0.25" away from fold lines
  • Spine: Allow extra margin for binding if applicable
Color & Image Guidelines
  • Black Text: Use 100K for pure black text
  • Rich Black: Use C40 M30 Y30 K100 for solid areas
  • Image Quality: Minimum 300 DPI for photos
  • Color Matching: Use Pantone colors when possible
Submission Requirements
  • File Naming: Use descriptive, clear file names
  • Proofing: Submit low-res PDF for approval
  • Final Files: High-res print-ready files required
  • Revisions: Include revision notes with resubmissions
